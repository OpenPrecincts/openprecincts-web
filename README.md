Getting Started
---------------

* Install [pipenv](https://pipenv.readthedocs.io/en/latest/)
* ``pipenv install``
* Install [npm](https://www.npmjs.com/)
* ``npm install``
* ``pipenv run ./manage.py migrate``
* ``pipenv run ./manage.py init_data --groups``
* ``pipenv run ./manage.py runserver``
